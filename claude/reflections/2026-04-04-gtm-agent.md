# Reflect: GTM Agent — 20 Generations in 3 Days
Date: 2026-04-04
Grade: 7.5/10
Project: ~/code/gtm-agent

## Summary
Complete redesign of gtm.tangle.tools from marketing page to functional AI GTM operating system. 20 generations shipped. Security audit found 11 issues, all fixed. Architecture refactored from 723-line god file to 4 focused modules.

## Top Learnings
1. Fix the library (sandbox-ui), don't bypass it with custom components
2. Tailwind v4 doesn't scan node_modules — library must ship pre-built CSS
3. Test E2E with curl on live deployment before declaring done
4. System prompt should be lean (~800 tokens) — agent reads vault via bash
5. Action over questions — agent's value is doing (42 tool calls, names competitors with pricing), not asking 5 questions

## Mistakes
- Created ChatInputCustom/ChatMessageCustom 3 times, reverted each time
- Deployed broken features 5 times (missing endpoint, wrong response structure, dead proxy, missing DB column, wrong model config)
- System prompt bloated to 4000 tokens before being cut
